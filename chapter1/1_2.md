## Exercise 1.9:

```scheme
(define (+ a b)
  (if (= a 0) 
      b 
      (inc (+ (dec a) b))))
; using substitution model calculate (+ 4 5)
(+ 4 5)
(inc (+ (dec 4) 5))
(inc (+ 3 5))
(inc (inc (+ (dec 3) 5)))
(inc (inc (+ 2 5)))
(inc (inc (inc (+ (dec 2) 5))))
(inc (inc (inc (+ 1 5))))
(inc (inc (inc (inc (+ (dec 1) 5)))))
(inc (inc (inc (inc (+ 0 5)))))
(inc (inc (inc (inc 5))))
(inc (inc (inc 6)))
(inc (inc 7))
(inc 8)
9
; The process above is a linear recursive process. There are more information about the state
; than can be contained in the parameters.
```



```scheme
(define (+ a b)
  (if (= a 0) 
      b 
      (+ (dec a) (inc b))))

(+ (dec 2) (inc 7))

(+ (dec 1) (inc 8))
```
